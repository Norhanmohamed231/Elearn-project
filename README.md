# Elearn Platform

**Elearn** is an interactive educational platform designed to enhance the learning experience for both students and teachers. The platform offers a smooth and user-friendly interface where users can register, create profiles, manage courses, and participate in a dynamic learning environment.

---

## 🚀 Features

### 🔐 User Registration & Profiles
- Users can sign up as **teachers** or **students**.
- Each user has a customizable profile for a personalized experience.

### 📚 Course Management
- **Teachers** can create, edit, and manage their own courses.
- **Students** can browse available courses, enroll in them, and cancel enrollment when necessary.

### 📊 Dynamic Learning Experience
- Real-time tracking of enrolled students per course.
- Intuitive interface for exploring and joining courses with ease.
